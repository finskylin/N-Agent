## 任务

## 原始查询
{user_query}

## 完整分析内容
{full_analysis}

## 输出要求
输出一段精简的 Markdown 摘要，包含：
1. **核心结论**（2-3 句话，直接回答用户问题）
2. **关键数据**（3-5 个最重要的数据点，用列表展示）
3. **操作建议**（1 句话，具体可行）
4. **下一步**：根据分析内容，推测用户可能感兴趣的后续动作，以提问形式给出 2-3 个建议。

格式示例：
---
**您可能还想了解：**
- 是否需要对比同行业其他公司？
- 需要查看最近的资金流向吗？

约束：
- 总长度不超过 800 字
- 不包含免责声明、数据来源表格
- 不包含图片或报告下载链接
- 保留关键数字和指标
- 若完整内容包含数据表格（天气预报逐日数据、股票行情、财务数据等），在摘要末尾追加一行：`> 完整数据表格已附在报告文件中，可下载查看`
- 使用{lang}输出
